Choisir ses barrettes de RAM

Mémoire volatile du PC gamer, les barrettes de RAM assurent la rapidité globale de la machine. Pour éviter que l’ordinateur ne manque de RAM et soit contraint de traiter les données par le disque dur, une quantité de 16 Go de RAM est idéale. Si votre budget est trop serré, 8 Go de RAM sont suffisants pour jouer dans la grande majorité des cas. Pour une efficacité optimale, une fréquence de 1600 MHz ou plus est recommandée pour la RAM. Par ailleurs, plus la latence CAS (délai d’accès colonne) est réduite, plus la mémoire sera rapide. Attention : les barrettes de RAM doivent être compatibles avec votre carte mère.
